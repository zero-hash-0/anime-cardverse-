## Description
Radar helps you monitor Solana wallet activity with a focus on airdrop safety and fast situational awareness.

Key features:
- Airdrop scanner for wallet token changes
- Risk scoring for suspicious claim patterns
- High-risk filtering and watchlist views
- Local alert controls and auto-scan support
- Security Center with biometric-gated controls
- Live Solana news pulse with rotating headlines and trending topics

Radar is built for users who want a cleaner way to monitor new token arrivals and reduce risky interactions.
